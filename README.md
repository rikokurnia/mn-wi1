# Mandora

> **Proof of work for the real world. Instant payout on Solana.**

Mandora is a Solana-based labor verification and instant payout protocol designed for Indonesia’s informal workforce.
We help agencies verify real-world field work and automate stablecoin payouts through programmable escrow infrastructure.

---

# 🚀 The Problem

Indonesia has more than **5.5 million informal gig workers**, yet many still face:

* Delayed payments
* Manual verification systems
* Fraudulent GPS/photo proofs
* Operational overhead from spreadsheets and payroll administration

Traditional workflows are slow, fragmented, and expensive.

---

# 💡 The Solution

Mandora transforms physical work into programmable payments.

Using geofenced verification and Solana-based settlement infrastructure, agencies can:

* Verify field work in real time
* Lock payouts into escrow
* Automatically release IDRX stablecoin payments once tasks are completed

---

# ⚙️ How It Works

### 1. Proof of Location

Workers submit GPS-based check-ins and photo proofs.

### 2. Escrow Executes

Agency funds are locked securely before tasks begin.

### 3. Automatic Payout

Verified tasks instantly trigger IDRX settlement on Solana.
